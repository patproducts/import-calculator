# PAT Products Import Costing and Suggested Pricing Calculator

A web-based calculator for determining landed costs and suggested pricing for imported goods.

## Features

- **Multi-currency support** (USD/EUR with exchange rates)
- **Automatic fee calculations** (Insurance @ 0.164%, MPF @ 0.4714%)
- **Comprehensive duty tracking** (Base duty, Trump tariffs, ADD/CVD, anti-dumping, countervailing)
- **Flexible duty inputs** (percentage-based or per-kg rates)
- **Package-level costing** (calculates cost per kg, per lb, and per package)
- **Profit margin suggestions** (7 pricing scenarios from 5% to 40% margin)
- **Multi-warehouse support** (NJ, IL, TX, NH, ME)
- **Timestamped calculations** for record keeping

## Usage

This calculator is designed for PAT Products import operations covering chemicals, plastics, and food ingredients from China, Germany, Italy, and other countries.

All calculations are performed client-side - no data is stored or transmitted.
